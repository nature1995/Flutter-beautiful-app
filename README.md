# Getting Start

## Hello World

One of the simplest Flutter apps, just a widget! As an example below: Pass a widget to the `runApp` function:
```dart
import 'package:flutter/material.dart';

void main() {
  runApp(
    new Center(
      child: new Text(
        'Hello, world!',
        textDirection: TextDirection.ltr,
      ),
    ),
  );
}
```
